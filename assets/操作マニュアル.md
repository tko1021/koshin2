# DIPS更新修了者 管理アプリ（クラウド版）操作マニュアル

登録更新講習機関（機関コード 0188）の修了者を管理し、DIPS登録用CSVと修了証明書を
作成するアプリです。ブラウザからアクセスして使います。

---

## 1. ログイン
- 配布されたURLをブラウザで開く
- **パスワード**を入力してログイン（個人情報保護のため）

## 2. 画面の流れ
左のサイドバーでページを切り替えます。

```
① 修了者を登録 ──→ ②一覧・編集（修正/削除）
        │
        ├──→ ③ DIPS_CSV作成（登録用CSVをDL）──→ DIPSへアップロード
        │
        ├──→ ⑤ 修了証明書発行（PDFをDL）
        │
        └──→ ④ DIPS訂正（登録済みの上書き修正/無効化）
```

## 3. 修了者を登録（①）
フォームに入力して「登録する」。
- 講習区分・等級区分・受講有無・状態は選択式
- **担当講師・限定解除事項**（区分ごとに 基本/目視外飛行/昼間飛行/25kg未満）も登録可
- 証明書番号・有効期限はプレビュー表示（自動採番）

## 4. 一覧・編集（②）
登録済みを選んで内容を修正、または削除。証明書の担当講師・限定解除もここで修正します。

## 5. DIPS登録用CSVを作成（③）
「DIPS登録用CSV を発行」→ 公式17列CSVをダウンロード。
- **未発行の人だけ**が出ます（発行済みは履歴で自動除外）
- セルフテスト（BOM/17列/CRLF等）合格を確認
- ダウンロードしたCSVをDIPSにアップロード

## 6. 修了証明書を発行（⑤）
対象を選んで「修了証明書を発行」→ PDFをダウンロード（全員分はZIP）。
公式様式1・印影入り。担当講師・限定解除は登録内容が反映されます。

## 7. DIPS訂正（④）
発行済みの登録を直すとき。
- **上書き修正**：先に「一覧・編集」で内容を直し、ここで上書き修正CSVを発行
- **無効化**：登録を取り消すCSVを発行（確認チェックあり）

---

## 注意
- データはGoogleスプレッドシートに保存されます（クラウド）
- CSV・PDFは画面からダウンロードして使います（端末に保存）
- 操作で困ったら管理者へ連絡してください
